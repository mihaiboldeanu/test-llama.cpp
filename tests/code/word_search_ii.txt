Find all words from a dictionary present in a 2D board.

def find_words(board: list[list[str]], words: list[str]) -> list[str]:

Requirements:
- A word can be formed by sequentially adjacent cells (up/down/left/right)
- Each cell may be used at most once per word
- Return all distinct words from `words` that exist in the board
- Return results in any order
- Use an efficient Trie/backtracking solution
- If `board` is empty or `words` is empty, return `[]`
- The same word should appear at most once in the output even if it appears multiple times in `words`

Examples:
find_words([["o","a","a","n"],["e","t","a","e"],["i","h","k","r"],["i","f","l","v"]], ["oath","pea","eat","rain"]) -> ["oath","eat"]
